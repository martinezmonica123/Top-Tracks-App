Description

The Top Tracks app is currently in development.

The goal of the Top Tracks app is to provide Spotify users with their top tracks and top artists information, create a playlist of the top tracks, and provide a a visual representation of the change in top tracks overtime.

Technologies

  • Node.js
  • Passport.js
  • Spotify Web API Node
  • Bootstrap
